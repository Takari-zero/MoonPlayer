# Moon播放器 产品说明书

## 1. 产品定位

Moon播放器 是一个 Android 本地媒体播放器，聚焦用户设备中的本地内容管理和播放。

核心方向：

- 本地视频播放
- 本地音乐播放
- TXT 小说听书
- 本地文件夹和文件管理
- 简洁、稳定、无干扰的播放器体验

明确不做：

- 在线视频
- 在线音乐
- 在线书源
- 登录账号
- 云同步
- 社交功能
- 广告 SDK
- 推荐信息流

## 2. 底部导航

底部导航固定四栏：

- 视频
- 音乐
- 小说
- 我的

规则：

- 底部导航只出现在一级页面。
- 播放页、详情页、设置页、听书页等二级页面不显示底部导航。
- 不新增一级 Tab，除非产品方向明确调整。

## 3. 视频模块

视频模块定位为本地视频库和本地视频播放器。

应包含：

- 自动扫描本地视频
- 手动添加视频文件夹
- 视频文件夹列表
- 视频文件夹详情页
- 视频播放页
- 视频缩略图
- 视频时长
- 视频进度记忆
- 最近播放 / 继续播放
- 上一集 / 下一集
- 默认横屏播放
- 亮度 / 音量上下滑动
- 左右滑动快进 / 快退
- 倍速播放
- 画面比例 / 适应屏幕
- 循环播放
- 音轨入口
- 字幕入口和外挂字幕入口
- 删除 / 多选删除
- 从列表移除 / 永久删除
- 自动扫描文件夹或文件的隐藏机制，如果后续已实现

视频模块不要做：

- 在线视频
- 弹幕
- 云端字幕搜索
- 账号同步播放进度
- 视频后台播放
- 视频推荐

## 4. 音乐模块

音乐模块定位为本地音乐播放器。

应包含：

- 自动扫描本地音乐
- 手动添加音乐文件夹
- 曲目列表
- 文件夹列表
- 后台播放
- MediaSessionService
- 通知栏控制
- 锁屏控制
- 迷你播放器，并且只在音乐主页显示
- 音乐播放页
- 播放队列
- 播放模式：顺序 / 随机 / 单曲循环 / 列表循环
- 上一曲 / 下一曲
- 均衡器
- 从列表移除
- 永久删除失败时自动隐藏
- `hidden_audio_uris_json`
- 设置页恢复隐藏音乐

音乐模块不要做：

- 在线音乐
- 歌词联网搜索
- 账号歌单同步
- 推荐算法
- 云盘音乐
- 音乐播放进度记忆，除非后续重新确认

## 5. 小说模块

小说模块定位为本地 TXT 小说书架和听书工具。

应包含：

- 单本 TXT 导入
- 多选 TXT 导入
- TXT 文件夹导入
- 小说书架
- TXT 导入记录持久化
- 删除 / 多选删除
- 听书页
- TextToSpeech 朗读
- 语速 / 音调
- 上一段 / 下一段
- 听书进度
- 书架状态：未开始 / 已听 X% / 已完成
- TTS 不可用时，引导用户安装语音数据或打开系统语音设置

小说模块不要做：

- 在线书源
- 章节智能抓取
- AI 配音
- 小说后台朗读，除非后续单独批准
- 账号同步

## 6. 我的 / 设置模块

我的页是个人入口和管理入口，不是单纯设置页。

应包含：

- App 信息
- 扫描管理
- 播放设置
- 小说设置
- 数据管理
- 恢复隐藏音乐
- 清除播放进度
- 清除小说导入记录
- 关于 Moon播放器

设置页应从“我的”进入，不作为底部一级 Tab。

## 7. 数据持久化

当前优先使用轻量本地持久化，避免过早引入数据库。

应记录：

- 手动添加文件夹
- 视频播放进度
- 小说导入 TXT
- 小说听书进度
- `hidden_audio_uris_json`
- hidden video / folder 记录，如果后续已实现
- 最近播放视频
- 最近播放音乐，如果用于迷你播放器或播放队列恢复

不应保存：

- 视频缩略图 Bitmap
- 大量扫描结果缓存
- 文件真实内容
- TXT 全文内容
- 用户账号数据
- 网络数据

## 8. 删除策略

删除分为两类。

### 从列表移除

- 不删除真实文件。
- 必须写入隐藏记录。
- 自动扫描时必须过滤隐藏记录，避免重启后重新出现。
- 需要同步更新 UI、播放队列、最近播放、进度状态。

### 永久删除

- 尝试通过系统允许的 ContentResolver / SAF 能力删除真实文件。
- 删除前必须二次确认。
- 删除成功后同步更新 UI、播放队列、进度状态和 DataStore。
- 删除失败时提示权限问题。
- 对音乐等系统限制明显的场景，可以降级为“从列表移除”，并明确提示用户。

删除操作不得静默失败。

## 9. 后续开发顺序

建议按以下顺序推进：

1. 阶段 1：视频模块 UI 与功能收口
2. 阶段 2：音乐模块 UI 与功能收口
3. 阶段 3：小说听书 UI 与 TTS 稳定性收口
4. 阶段 4：我的 / 设置页整理
5. 阶段 5：稳定性测试与发布准备

每个阶段应先完成单模块闭环，再进入下一个模块。

## Video Folder Detail Navigation Exception

- Video folder detail is an explicit exception to the general secondary-page bottom-navigation rule.
- It reuses `MoonBottomNavigationBar` so video home and folder detail bottom bars stay visually identical.
- `Video` stays highlighted in folder detail.
- `Music`, `Novel`, and `Me` are real top-level navigation targets.
- Tapping the already-selected `Video` tab in folder detail does not force navigation back to video home; use the top-left back button.
- The video player page remains unchanged.
