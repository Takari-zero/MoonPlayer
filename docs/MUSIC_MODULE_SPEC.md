# Moon播放器 音乐模块说明

## 1. 模块定位

音乐模块是本地音乐播放器，重点是曲目管理、后台播放、通知栏控制和稳定播放队列。

目标：

- 自动扫描本地音乐。
- 支持手动添加音乐文件夹。
- 曲目列表为默认首页。
- 支持后台播放、通知栏控制和锁屏控制。
- 支持删除、隐藏和恢复隐藏音乐。

不做：

- 在线音乐
- 在线歌词搜索
- 云歌单同步
- 登录
- 推荐算法
- 云盘音乐

## 2. 音乐主页结构

音乐主页是底部导航“音乐”的一级页面。

页面结构：

- 顶部栏
  - 标题：音乐
  - 副标题：本地音乐
  - 搜索
  - 更多
- Tab
  - 曲目
  - 列表
  - 专辑
  - 艺人
  - 文件夹
- 内容区
  - 默认显示曲目列表
  - 文件夹 Tab 保留按文件夹展示
- 迷你播放器
  - 只在音乐主页显示
  - 不在视频、小说、我的或播放页显示

## 3. Tab 结构

本阶段必须实现：

- 曲目
- 文件夹

可占位：

- 列表
- 专辑
- 艺人

Tab 规则：

- 不竖排。
- 不换行。
- 文案短，推荐：曲目 / 列表 / 专辑 / 艺人 / 文件夹。
- 未实现 Tab 必须显示空状态或 Toast “后续实现”。

## 4. 曲目列表

曲目列表展示扁平化音频文件。

来源：

- MediaStore 自动扫描音频。
- 手动添加音乐文件夹扫描出的音频。

展示字段：

- 封面占位或音乐图标。
- 歌名，优先去掉扩展名。
- Artist，读取不到显示 Unknown。
- 文件大小，可选。
- 右侧三点菜单。

规则：

- 按 uri 去重。
- 过滤 `hidden_audio_uris_json`。
- 点击歌曲直接进入播放。
- 点击歌曲从头播放，不恢复上次播放位置。
- 不显示“上次播放到 xx:xx”。

## 5. 迷你播放器

迷你播放器只在音乐主页显示。

应显示：

- 当前歌曲名。
- Artist。
- 播放 / 暂停按钮。
- 队列入口。
- 简单进度。

规则：

- 控制 MediaSessionService 中的全局播放器。
- 点击进入 AudioPlayerScreen。
- 没有当前歌曲时不显示。
- 离开 AudioPlayerScreen 后音乐继续播放。

## 6. 音乐播放页

音乐播放页控制后台 Service，不创建独立 ExoPlayer。

应包含：

- 返回按钮。
- 歌名。
- Artist。
- 唱片视觉或封面占位。
- 当前时间。
- 总时长。
- 进度条。
- 播放 / 暂停。
- 上一曲 / 下一曲。
- 播放模式。
- 均衡器。
- 更多。

规则：

- 进度条使用连续轨道。
- Thumb 纯圆形，无黑边。
- 拖动中预览，松手后 seek。
- 音乐不记录播放进度。
- 页面返回不停止播放。

## 7. 后台播放 Service

音乐后台播放使用 Media3 MediaSessionService。

Service 职责：

- 持有全局 ExoPlayer。
- 持有 MediaSession。
- 支持后台播放。
- 支持通知栏控制。
- 支持锁屏控制。
- 支持播放 / 暂停 / 上一曲 / 下一曲。
- 支持播放队列。
- 支持播放模式。

不得将视频播放迁移到音乐 Service。

## 8. 通知栏 / 锁屏控制

通知栏应显示：

- 歌曲名。
- Artist。
- 播放状态。

通知栏按钮：

- 上一曲
- 播放 / 暂停
- 下一曲

Android 13+：

- 需要通知权限。
- 用户拒绝权限时不得崩溃。
- App 内播放仍可继续。

## 9. 播放队列

点击曲目时：

- 队列 = 当前曲目列表。
- 当前 index = 点击歌曲。
- 从头播放。

上一曲：

- 当前播放超过 3 秒，先回到当前歌曲开头。
- 否则切换上一首。

下一曲：

- 根据播放模式决定下一首。

删除当前歌曲：

- 从队列移除。
- 如果正在播放，可停止或切换下一首。
- 迷你播放器和通知栏状态必须更新。

## 10. 播放模式

支持：

- 顺序
- 随机
- 单曲循环
- 列表循环

规则：

- 单击播放模式按钮顺序切换模式。
- 长按播放模式按钮弹出完整模式选择。
- 图标和文字必须同步当前模式。
- 切换歌曲时从头播放。

## 11. 均衡器

均衡器为音乐播放页功能。

应包含：

- 开关。
- 预设：默认 / 古典 / 舞曲 / 人声 / 摇滚 / 金属。
- 多段竖向滑杆。
- 重置。
- 完成。

UI 规则：

- 蓝色基调。
- 滑杆细。
- Thumb 圆形。
- BottomSheet 高度合理，不占满屏。
- 内容紧凑，不留过大空白。

## 12. 删除 / 隐藏音乐

删除分为：

- 从列表移除
- 永久删除文件

从列表移除：

- 写入 `hidden_audio_uris_json`。
- 从 `audioFiles` 移除。
- 从 `audioFolders` 移除。
- 从播放队列移除。
- 重启 App 后不再出现。

永久删除：

- 必须二次确认。
- 尝试删除真实文件。
- 删除成功后从 UI、队列和 DataStore 移除。
- 删除失败时提示“系统不允许直接删除该文件，已为你从列表移除。”并写入隐藏记录。

设置页：

- 必须提供“恢复隐藏音乐”。
- 点击后清空 `hidden_audio_uris_json` 并重新扫描音乐。

## 13. 音乐进度策略

音乐不记录播放进度。

规则：

- 点击歌曲始终从头播放。
- 不向 DataStore 写入音乐进度。
- App 启动时忽略旧音乐进度。
- 设置页清除播放进度时可顺手清除旧音乐进度。
