# Moon播放器 UI 规范

## 1. 总体风格

Moon播放器 的 UI 应体现成熟本地播放器的质感。

总体方向：

- 以深色蓝黑主题为主。
- 小说听书页可以使用浅色阅读风格，但需要和整体产品气质协调。
- 蓝色作为主要强调色。
- 减少粉色 / 紫色大面积使用。
- 避免默认 Material Demo 感。
- 避免功能按钮堆叠。
- 避免列表项和卡片过大。
- 所有界面都应重视尺寸、居中、间距和层级。

## 2. 颜色规范

建议色值：

| Token | Color | 用途 |
| --- | --- | --- |
| `background` | `#05070D` | 页面背景 |
| `surface` | `#101722` | 顶部栏、底部栏、弹层背景 |
| `surfaceSoft` | `#151E2B` | 次级容器 |
| `card` | `#182233` | 卡片、文件夹图标背景 |
| `cardSoft` | `#1D2A3D` | 轻量卡片 |
| `primary` | `#4D8DFF` | 主强调色 |
| `primarySoft` | `#8AB6FF` | thumb、弱强调 |
| `textPrimary` | `#F5F7FA` | 主文字 |
| `textSecondary` | `#A8B2C2` | 副文字 |
| `textMuted` | `#6F7A8A` | 弱文字 |
| `divider` | `#263244` | 分割线 |
| `danger` | `#F97066` | 删除、危险操作 |

原则：

- 顶部区域、内容区域、底部导航背景应协调，避免明显割裂。
- 危险操作只使用红色，不使用蓝色伪装为普通操作。
- 强调色只用于关键状态、选中态和主操作。

## 3. 顶部栏规范

### 主页面

主页面包括：视频、音乐、小说、我的。

规范：

- 可显示标题、副标题和功能按钮。
- 功能按钮不能重复出现在内容区。
- 搜索默认用图标入口，不常驻占一整行。
- 功能多时应收纳进三点菜单或工具面板。
- 顶部栏不应挤压主要内容。

### 二级页面

二级页面包括文件夹详情、设置、听书、播放器等。

规范：

- 左侧统一使用返回箭头。
- 中间显示标题。
- 右侧显示功能入口。
- 标题保持视觉居中。
- 如果右侧无按钮，应保留等宽占位，避免标题偏移。

### 播放页面

规范：

- 顶部只保留返回、标题、三点或少量明确入口。
- 不要堆太多文字按钮。
- 视频播放页顶部栏应半透明。
- 音乐播放页标题区域应简洁，不放无关入口。
- 小说听书页顶部应突出书名和听书状态。

## 4. 底部导航规范

底部导航固定四栏：

- 视频
- 音乐
- 小说
- 我的

规范：

- 选中态使用蓝色。
- 背景和页面背景统一。
- 不要使用突兀的大色块。
- 不要新增“设置”一级 Tab。
- 二级页面和播放页不显示底部导航。

## 5. 列表规范

### 视频列表

结构：

- 缩略图
- 标题
- 时长
- 大小 / 上次播放进度
- 三点更多

规范：

- 缩略图圆角统一。
- 时长标签放在缩略图右下角。
- 标题最多 2 行。
- 列表项紧凑，不要过高。
- 当前播放或最近播放使用蓝色弱提示。

### 音乐列表

结构：

- 封面占位或音乐图标
- 歌名
- Artist / size
- 三点更多

规范：

- 不显示音乐播放进度记忆。
- 点击歌曲从头播放。
- 删除和隐藏状态必须同步更新列表。

### 小说书架

结构：

- 3 列书本卡片
- TXT 角标
- 小说名
- 进度状态

规范：

- 第一格为导入入口。
- 普通书籍支持长按删除。
- 书架只展示用户明确导入的 TXT。

## 6. 网格规范

视频主页网格：

- 3 列。
- Item 居中。
- 文件夹图标不宜过大。
- 文件夹名居中。
- 视频数量居中。
- 角标放在文件夹图标右上角。
- 角标使用蓝色，不使用粉色。
- 左右边距统一。
- 最后一行不能被 FAB 遮挡。

建议尺寸：

- 文件夹卡片宽度：86dp - 100dp。
- 文件夹卡片高度：64dp - 76dp。
- 水平间距：18dp - 24dp。
- 垂直间距：18dp - 24dp。

## 7. 播放页规范

### 视频播放页

规范：

- 默认横屏。
- 顶部 / 底部控制层半透明。
- 顶部只保留必要入口。
- 进度条细蓝色。
- Thumb 使用圆形。
- 右上功能面板为工具面板。
- 支持左右滑动快进 / 快退。
- 支持上下滑动亮度 / 音量。
- 控制层自动隐藏。
- 面板打开时不应自动消失，需设置完成、关闭或返回手势关闭。

### 音乐播放页

规范：

- 唱片视觉要精致。
- 进度条使用 Canvas 连续轨道。
- Thumb 纯圆形，无黑边。
- 播放按钮使用蓝色系。
- 播放模式菜单靠近按钮弹出。
- 不记录音乐播放进度。

### 小说听书页

规范：

- 更像阅读 / 听书页。
- 文本区域是重点。
- TTS 状态提示清晰。
- 语速 / 音调不常驻展开。
- TTS 不可用时必须给出明确引导。

## 8. 菜单、Dialog、BottomSheet 规范

规范：

- 使用大圆角。
- 深色背景。
- 图标 + 文案。
- 危险操作使用红色。
- 删除必须二次确认。
- 不要出现无效按钮。
- 不能实现的功能必须 Toast 明确说明。
- 工具面板不要占满屏，除非它本身是完整设置页。
- 点击外部关闭仅适用于临时菜单；播放器设置面板可保持常驻直到用户主动关闭。

## 9. 进度条和 Slider 规范

统一样式：

- 所有 thumb 使用圆形。
- 不使用竖线 thumb。
- `active`: `#4D8DFF`
- `inactive`: `#2E3445`
- `thumb`: `#8AB6FF`
- 轨道要细，不要厚重。
- 拖动中只预览。
- 松手后再执行 seek。
- Thumb 不应有黑色外圈、断层或阴影。

适用范围：

- 视频进度条
- 音乐进度条
- 小说听书进度条
- 均衡器滑杆
- 速度、音调等调节滑杆

## 10. 禁止事项

- 不要无计划新增功能。
- 不要把功能按钮重复放在顶部和内容区。
- 不要默认大块卡片。
- 不要让列表 item 过大。
- 不要让 UI 不居中。
- 不要让弹窗占满屏，除非是明确页面。
- 不要使用默认 Demo 风格控件堆页面。
- 不要在一级页面常驻大搜索框。
- 不要把未实现功能做成假成功。

## Video Folder Detail Bottom Navigation Exception

- General secondary pages still avoid bottom navigation unless a module explicitly defines an exception.
- Video folder detail is now an explicit exception: it shows the same bottom navigation as video home by reusing `MoonBottomNavigationBar`.
- The bottom bar must match video home exactly; do not create a separate similar-looking implementation.
- In video folder detail, tapping the selected `Video` tab does not force navigation back to video home. Use the top-left back button to return.
- The list content and lower-right play FAB must avoid the bottom navigation bar.

## 音乐模块近期 UI 状态

最近完成提交：`b77cd10 feat(media): refine music player and unify search`。

- 音乐主页采用紧凑高信息密度布局，顶部显示“音乐 / 本地音乐”，右侧为搜索和更多。
- 音乐搜索框在顶部栏内展开，不新增搜索行，不撑高顶部栏；内部无搜索图标、无 X。
- 点击搜索结果只播放并退出搜索；点击空白区域退出搜索。
- “我喜欢 / 最近播放 / 音乐文件夹”应保持紧凑入口形态；没有真实数据时显示真实空状态，不写假内容。
- “我喜欢 / 最近播放 / 音乐文件夹”和“全部歌曲 / 歌曲数量”固定，只有歌曲列表滚动。
- 主页主体是真实音频列表；歌曲行应保持紧凑，当前播放歌曲使用轻量高亮。
- 无封面时使用统一默认音乐图标 / 月相方向视觉，不显示破图。
- 迷你播放器位于音乐主页底部导航栏上方，外层包裹区域透明，内部控制卡片保留深色圆角卡片。
- 迷你播放器支持上一曲 / 播放暂停 / 下一曲，进度线位于内部控制器底部并支持点击 / 拖动 seek。
- `RotatingMusicThumb` 用于左侧旋转小方块，当前方向为紫色月亮 / 月相，小方块整体慢速旋转。
- 迷你播放器不得修改 `MoonBottomNavigationBar` 的尺寸、选中态、图标和文字样式。
- 音乐播放页第一版 UI 已落地：沉浸式展示，不显示底部导航栏；顶部只有下箭头和更多；使用月相黑胶盘方向；进度和基础播放控制接真实状态。
- 音乐播放页底部功能入口中，歌词、A-B、定时、收藏等仍可能是 Toast / 占位 / 后续能力，不要写成完整功能。
- 跨模块顶部栏搜索使用 `MoonInlineSearchField` 对齐：音乐、视频、小说均为顶部栏内展开，搜索按钮和更多按钮不消失，内部无搜索图标和 X，不撑高顶部栏。
- 视频文件夹详情页搜索也应对齐这套顶部栏搜索样式。
- 底部导航图标已更新为书本 / 个人图标，但底部导航栏整体样式、高度、选中态和功能逻辑不变。
